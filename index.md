# 선(한) 결제 - 서비스 소개

> 투명하고 따뜻한 선결제 기부 문화를 만들어갑니다.

---

## 서비스 소개

### 선(한) 결제란?

**선(한) 결제**는 야외 집회나 봉사활동 참가자들을 금전적으로 지원하는 **선결제 기부 문화**를 투명하고 건강하게 관리할 수 있는 모바일 앱 서비스입니다.

### 왜 필요한가요?

최근 집회나 봉사활동 참가자들을 응원하기 위해 인근 카페나 식당에 미리 선결제를 해두는 아름다운 기부 문화가 확산되고 있습니다. <br><br>
하지만 일부에서 선결제 금액을 제대로 사용하지 않거나, 후원자가 자신의 기부가 어떻게 사용되었는지 확인하기 어려운 문제가 있었습니다.

**선(한) 결제**는 이러한 문제를 해결합니다:

- **투명한 기록**: 모든 선결제와 사용 내역이 앱에 기록됩니다
- **QR 코드 결제**: 악용을 방지하는 안전한 결제 시스템
  - **일일 사용 제한**: 수혜자는 일일 3회까지 선결제를 사용할 수 있습니다.
  - **위치 기반 검증**: 매장 위치에서만 결제 처리가 가능합니다.
- **실시간 확인**: 후원자가 자신의 선결제가 어떻게 사용되었는지 바로 확인
- **감사 인사**: 수혜자가 후원자에게 직접 감사의 마음을 전달

### 세 가지 사용자 유형

| 유형                        | 설명                                            |
| --------------------------- | ----------------------------------------------- |
| **선결제자 (Benefactor)**   | 선결제 후원자                                   |
| **수혜자 (Beneficiary)**    | 선결제 수혜자                                   |
| **매장 업주 (Store Owner)** | 선결제를 수락하고 관리하는 매장(카페/식당) 업주 |

### 로그인

앱을 실행하면 로그인 화면이 나타납니다.

![로그인 화면]({{ site.baseurl }}/images/login.png)

**로그인 방법:**

1. **카카오 로그인** - 카카오 계정으로 간편 로그인 (Android/iOS)

**유저 타입 선택:**
로그인 시 본인의 역할에 맞는 유저 타입을 선택하세요:

- **매장 업주**: 매장에서 선결제를 관리하시는 분

> 💡 **TIP**: 유저 타입은 나중에 프로필에서 변경할 수 있습니다.

## 사용 방법

### 선결제자 (Benefactor) - 후원하기

선결제자는 봉사자나 집회 참가자들을 위해 매장에 미리 결제해 두는 후원자입니다.

#### 1단계: 선결제 생성하기

홈 화면에서 **"신규 선결제 정보 입력"** 버튼을 눌러 새로운 선결제를 등록합니다.

![선결제 생성 화면]({{ site.baseurl }}/images/benefactor_1.png)

**입력 항목:**

- **선결제 가게**: 매장 이름을 검색하여 선택
- **선결제 금액**: 후원할 총 금액 (예: 400,000원)
- **선결제 일자**: 선결제 유효 기간
- **결제 건당 차감 금액**: 1회 사용 시 차감될 금액 (예: 3,000원)
- **남기는 메시지**: 수혜자에게 전할 응원 메시지 (선택)
- **대표 이미지**: 선결제에 첨부할 이미지 (선택)

#### 2단계: 선결제 관리하기

생성한 선결제는 홈 화면에서 목록으로 확인할 수 있습니다.

![선결제 상세 화면]({{ site.baseurl }}/images/benefactor_2.png)

**선결제 상태:**

- **준비중**: 아직 활성화되지 않은 상태 (수정/삭제 가능)
- **진행중**: 수혜자가 사용할 수 있는 상태
- **완료**: 잔액이 모두 소진되었거나 기간이 종료된 상태

**확인할 수 있는 정보:**

- 선결제 매장 정보 및 연락처
- 총 선결제 금액 및 잔여 금액
- 사용 내역
- 수혜자가 보낸 감사 인사

---

### 수혜자 (Beneficiary) - 혜택 받기

수혜자는 선결제 혜택을 받아 무료로 음식이나 음료를 제공받을 수 있습니다.

#### 1단계: 내 근처 선결제 찾기

앱을 실행하면 **현재 위치를 기준으로 가까운 선결제 목록**이 자동으로 표시됩니다.

![수혜자 홈 - 목록 보기]({{ site.baseurl }}/images/beneficiary_1.png)

**화면 구성:**

- **검색 반경 설정**: 1km ~ 30km 사이에서 선택
- **지도/목록 전환**: 지도 보기와 목록 보기 전환 가능
- **선결제 카드**: 매장명, 주소, 사용 가능 금액, 거리 표시

![수혜자 홈 - 지도 보기]({{ site.baseurl }}/images/beneficiary_2.png)

지도 보기에서는 선결제가 등록된 매장의 위치를 한눈에 확인할 수 있습니다.

#### 2단계: 선결제 상세 확인

원하는 선결제를 선택하면 상세 정보를 확인할 수 있습니다.

![선결제 정보 상세]({{ site.baseurl }}/images/beneficiary_3.png)

**확인할 수 있는 정보:**

- 매장 이름 및 주소
- 사용 가능 금액 (잔액)
- 선결제자 정보 및 응원 메시지
- 감사 인사 보내기 버튼

#### 3단계: QR 코드로 결제하기

**"QR 코드 생성"** 버튼을 누르면 결제용 QR 코드가 생성됩니다.

![QR 코드 생성]({{ site.baseurl }}/images/beneficiary_4.png)

**QR 코드 사용 방법:**

1. 결제 금액을 확인합니다 (기본값: 건당 차감 금액)
2. 생성된 QR 코드를 매장 직원에게 보여주세요
3. 매장 직원이 QR 코드를 스캔하면 결제가 완료됩니다

**주의사항:**

- QR 코드는 **5분간 유효**합니다 (보안을 위해)
- 일일 사용 제한: **계정당 3회, 디바이스당 5회**
- 사용 후 잔액에서 차감됩니다

#### 4단계: 감사 인사 보내기

선결제를 사용한 후, 선결제자에게 감사의 마음을 전해보세요!

선결제 상세 화면에서 **"감사 인사 보내기"** 버튼을 눌러 텍스트와 사진으로 감사 인사를 작성할 수 있습니다.

### 매장 업주 (Store Owner) - 매장 운영

매장 업주는 선결제를 관리하고 QR 코드 결제를 처리합니다.

#### 1단계: 매장 등록하기

처음 앱을 사용하시면 매장 등록 안내가 표시됩니다.

![매장 등록 안내]({{ site.baseurl }}/images/store_owner_1.png)

**매장 등록 방법:**

1. **"매장 검색하기"** 버튼을 누릅니다
2. 카카오맵에서 본인의 매장을 검색합니다
3. 검색 결과에서 매장을 선택하면 자동으로 등록됩니다

> 💡 카카오맵에 등록된 매장 정보(상호명, 주소, 연락처, 위치)가 자동으로 연동됩니다.

#### 2단계: 선결제 관리하기

매장이 등록되면 해당 매장에 등록된 선결제 목록을 확인할 수 있습니다.

![매장 업주 홈]({{ site.baseurl }}/images/store_owner_2.png)

**화면 구성:**

- **매장 정보**: 매장명, 주소, 연락처, 카카오맵 링크
- **선결제 목록**: 진행중/완료 탭으로 구분
- **선결제 카드**: 선결제자명, 잔액, 총액, 선결제일, 최근 사용일

#### 3단계: QR 코드 스캔하여 결제 처리

수혜자가 QR 코드를 제시하면 **"QR 코드 스캐너"** 버튼을 눌러 스캔합니다.

**결제 처리 절차:**

1. QR 코드 스캐너를 실행합니다
2. 수혜자의 QR 코드를 스캔합니다
3. 결제 정보(선결제 ID, 결제 금액, 수혜자)를 확인합니다
4. **"결제 처리"** 버튼을 눌러 완료합니다
5. 선결제 잔액에서 자동으로 차감됩니다

> ⚠️ **보안 안내**: QR 스캔 시 매장 위치가 확인됩니다. 등록된 매장 위치에서만 결제 처리가 가능합니다.

## 이용 안내

### 사용 제한

선결제 악용을 방지하기 위해 다음과 같은 제한이 적용됩니다:

| 항목              | 제한                       |
| ----------------- | -------------------------- |
| 수혜자 일일 사용  | 계정당 3회, 디바이스당 5회 |
| QR 코드 유효 시간 | 생성 후 5분                |
| 매장 위치 검증    | QR 스캔 시 매장 위치 확인  |

### 선결제 상태 안내

| 상태   | 설명                                            |
| ------ | ----------------------------------------------- |
| 준비중 | 선결제자가 생성했지만 아직 활성화하지 않은 상태 |
| 진행중 | 수혜자가 사용할 수 있는 활성 상태               |
| 완료   | 잔액 소진 또는 유효기간 종료                    |

### 보안 안내

- 모든 결제 내역은 투명하게 기록됩니다
- QR 코드에는 암호화된 결제 정보가 포함됩니다
- 위치 기반 검증으로 원격 악용을 방지합니다
